
# node.js 설치확인

> $ node -v

노드의 설치 버전이 올바르게 설치되었는지 확인한다.

> $ npm -v

npm(노드 패키지 매니저)이 올바르게 설치되었는지 확인한다

> $ npm install -g npm

npm의 버전을 업데이트 한다. (맥과 리눅스의 경우 앞에 sudo를 입력하고 비밀번호를 입력해야 할 수도 있다.)
